# Spatia Sauna Visual Identity Guidelines

## Brand Essence
Spatia Sauna is a sanctuary for transformation. Our visual identity combines Scandinavian minimalism with contemporary luxury.

## Design Philosophy
- **Scandinavian Minimalism**: Clean lines, lots of whitespace, functional beauty
- **Luxury Wellness**: Premium materials, refined typography, intentional spacing
- **Modern Sophistication**: Contemporary transitions, smooth animations, premium interactions
- **Accessibility First**: High contrast, readable text, keyboard-navigable

## Color Psychology

### Primary: Teal-Blue (#1E93AB)
- Deep, sophisticated, trust-building
- Evokes water, calm, healing
- Premium and contemporary
- Works for CTAs and primary elements

### Accent: Vibrant Red (#E62727)
- Energy, vitality, passion
- Creates bold contrast against teal
- Used strategically for urgency and emphasis
- Represents warmth and transformation

### Background: Warm Cream (#F3F2EC)
- Off-white with warmth
- Reduces eye strain
- Premium, organic feeling
- Perfect for clean, spacious layouts

### Neutral: Soft Gray (#DCDCDC)
- Subtle borders and dividers
- Text hierarchy support
- Calming and professional
- Creates visual breathing room

## Color Application

### Primary Teal (#1E93AB)
- Main headings
- Primary navigation
- Primary buttons (on light backgrounds)
- Brand accent elements
- Hero section highlights

### Accent Red (#E62727)
- Call-to-action buttons
- Important highlights
- Pricing badges
- Hover states
- Alert/urgency elements
- Left borders on cards

### Warm Cream (#F3F2EC)
- Page backgrounds
- Light surface areas
- Subtle overlays
- Contrast for text

### Neutral Gray (#DCDCDC)
- Borders
- Subtle dividers
- Inactive elements
- Secondary text
- Form field borders

## Typography

### Font Stack
- **Display**: Playfair Display (400–900, italic) — Headlines, hero sections
- **Body**: Inter (300–800) — Body text, UI elements
- **Mono**: JetBrains Mono (400) — Code, technical labels

### Type Scale
| Class | Size | Usage |
|-------|------|-------|
| `text-display-xl` | 3–5rem | Hero headlines |
| `text-display-lg` | 2–3.5rem | Section titles |
| `text-display-md` | 1.5–2.25rem | Featured content |
| `text-heading-xl` | 2rem | Section titles |
| `text-heading-lg` | 1.5rem | Subsections |
| `text-body-lg` | 1.125rem | Introductions |
| `text-body-md` | 1rem | Default text |
| `text-label-lg` | 0.875rem | Labels, tags |

## Spacing System
8px base grid for consistent rhythm.

| Token | Value | Usage |
|-------|-------|-------|
| `spacing-1` | 4px | Tight spacing |
| `spacing-2` | 8px | Default gap |
| `spacing-4` | 16px | Section padding |
| `spacing-8` | 32px | Section margins |
| `spacing-16` | 64px | Major sections |

## Border Radius
| Token | Value | Usage |
|-------|-------|-------|
| `radius-sm` | 4px | Buttons, inputs |
| `radius-md` | 8px | Cards |
| `radius-lg` | 12px | Large cards |
| `radius-xl` | 16px | Featured elements |
| `radius-full` | 9999px | Pills, avatars |

## Shadows & Elevation
All shadows use primary teal tint for brand cohesion.

## Animation Strategy
- **Easing**: `cubic-bezier(0.22, 1, 0.36, 1)` (spring feel)
- **Duration**: 300–500ms for smooth interaction
- **GPU Acceleration**: `transform` and `opacity` only
- **Reduced Motion**: Always respect `prefers-reduced-motion`

## Golden Rules
1. **Whitespace is wealth** — Give content room to breathe
2. **One accent per section** — Don't overwhelm with red
3. **Typography carries emotion** — Choose weights deliberately
4. **Every animation has purpose** — Motion should enhance, not distract
5. **Accessibility is non-negotiable** — WCAG AA minimum

## Do's and Don'ts

### Do
- Use ample whitespace
- Pair serif headlines with sans-serif body
- Use teal as primary brand color
- Use red sparingly for emphasis
- Apply subtle shadows and glows

### Don't
- Overuse the accent red
- Add decorative elements without purpose
- Stretch or distort the logo
- Use colors outside the defined palette
- Create heavy, cluttered layouts
